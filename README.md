# 여신금융협회 면접 대비 뉴스 모니터링 텔레그램 봇

네이버 뉴스 검색 API로 여신금융 관련 뉴스를 수집하여 텔레그램으로 전송하는 봇입니다.
PC(Windows)에서 상시 실행하고, 모바일 텔레그램에서 `/news` 명령으로 뉴스를 받아볼 수 있습니다.

## 파일 구성

| 파일 | 설명 |
|------|------|
| `news_bot.py` | 메인 봇 스크립트 |
| `.env` | API 키 설정 (git 추적 제외) |
| `requirements.txt` | Python 의존성 |
| `sent_news.json` | 전송 완료 뉴스 해시 (자동 생성) |

## 사전 준비

### 1) 네이버 검색 API 키 발급

1. https://developers.naver.com 접속 → 로그인
2. Application → 애플리케이션 등록
3. 사용 API: **검색** 선택
4. 비로그인 오픈 API → WEB → 주소: `http://localhost`
5. 등록 완료 → **Client ID** / **Client Secret** 복사

### 2) 텔레그램 봇 토큰 & Chat ID

- BotFather에서 봇 생성 → 토큰 발급
- 봇에 `/start` 전송 후 Chat ID 확인

## 설치 및 실행

```bash
# 의존성 설치
pip install -r requirements.txt

# .env 파일에 API 키 입력
NAVER_CLIENT_ID=your_id
NAVER_CLIENT_SECRET=your_secret
TELEGRAM_BOT_TOKEN=your_token
TELEGRAM_CHAT_ID=your_chat_id

# 실행
python news_bot.py
```

## 텔레그램 명령어

| 명령어 | 설명 |
|--------|------|
| `/news` | 즉시 뉴스 수집 및 전송 |
| `/help` | 도움말 |

## 모니터링 키워드 (4개 영역)

| 영역 | 키워드 |
|------|--------|
| 🏛️ 협회 | 여신금융협회, 정완규 |
| 💳 업권 | 신용카드, 캐피탈, 할부금융, 리스, 신기술금융, 벤처투자, 여전채/카드채, ABS |
| 📋 정책 | 카드수수료, 여신전문금융업법, 스테이블코인(+카드사 영향) |
| 🔥 경쟁 | 간편결제 |

### 검색 최적화

- 네이버 검색 연산자 활용 (`""` 정확 일치, `+` 필수 포함, `-` 제외)
- 키워드당 20건 수집 후 30일 이내 기사만 필터링
- 제목 기반 중복 제거로 동일 기사 방지

## 주요 기능

- **네이버 뉴스 API 연동**: 키워드별 최신 뉴스 수집
- **검색 연산자 최적화**: 정확도 높은 검색 쿼리로 노이즈 최소화
- **30일 기간 필터**: 최근 1개월 이내 기사만 수집
- **중복 제거**: 해시 + 제목 기반 이중 중복 방지
- **카테고리 분류**: 협회/업권/정책/경쟁 4개 영역 자동 분류
- **텔레그램 폴링**: `/news` 명령 수신 시 즉시 수집·전송
- **보안**: 지정된 Chat ID만 응답, API 키는 `.env`로 분리

## 문제 해결

| 증상 | 해결 |
|------|------|
| `urllib.error.HTTPError: 401` | 네이버 API 키 확인 |
| 텔레그램 메시지 안 옴 | 봇 토큰/Chat ID 확인, `/start` 보냈는지 확인 |
| 뉴스가 계속 0건 | `sent_news.json` 삭제 후 재실행 |
